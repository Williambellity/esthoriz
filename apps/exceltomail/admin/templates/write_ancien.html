Principe de fonctionnement de exceltomail

Cette application permet d'envoyer, en grand nombre, des mails personnalisés. Le principe est simple. Dans un excel réserver la première colonne aux adresses mails des destinataires. Les colonnes suivantes constituent les variables telles que le nom d'une entreprise ou d'un personne.

Exemple : Soit la ième ligne de l'excel : (Colonne 1 : forum@est-horizon.com, Colonne 2 : Forum Est-Horizon, Colonne 3 : DUPONT, Colonne 4 : Madame). Soit le message envoyé suivant : 'Bonjour ***004*** ***003***, nous vous informons que votre entreprise, ***002***, est bien inscrite pour notre prochaine édition. Bien à vous.' . Le mail reçu sera : 'Bonjour Madame DUPONT, nous vous informons que votre entreprise, Forum Est-Horizon, est bien inscrite pour notre prochaine édition. Bien à vous.' "

Remarque : la première ligne de l'excel est réservée aux titres (email, nom, prénom, ...).

Astuce : s'envoyer quelques mails tests au préalable.
